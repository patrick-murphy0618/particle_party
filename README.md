Particle Party:

A Python-based simulation that sends a single particle on a random path. The particle continues its party until it becomes trapped by its own path. Party on!


Features:

- User sets how many parties (iterations) are run
- Real-time visualization of the particles path
- Data of path collected and basic analysis provided
- Dataframe for advanced analysis returned


Getting Started (Installation)
