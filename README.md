# 🥳 Particle Party

A Python-based simulation that sends a single particle on a random path. The particle continues its party until it becomes trapped by its own path. **Party on!**

![Particle Party Animation](path/to/your/placeholder_image.gif)

## 🌟 Features
* **Customizable:** User sets how many "parties" (iterations) are run.
* **Real-time:** Live visualization of the particle's trajectory.
* **Data-Rich:** Automatic collection and basic analysis of path data.
* **Analysis Ready:** Returns a clean Pandas DataFrame for advanced data science work.
